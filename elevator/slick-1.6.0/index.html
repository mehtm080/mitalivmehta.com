This is a catalog for six compositions that I created using HTML, CSS and jQuery. The organizing principle here is an elevator which allows the user to gain access to the different floors (Levels- LL, 1, 2, 3, 4, and 5).
The lower level is my composition showing underwater. The first floor is land where Mickey Mouse and Minnie Mouse are shown rolling their eyes and a backdrop showing their home. The third floor is a composition of the sky where the color of the sky is changing and the birds are flying. (pure HTML and CSS, images have not been used in the first three compositions). The fourth floor shows a higher altitude with Santa Clause and falling snowflakes on hovering over them (use of jQuery). I have depicted a scene from the story of “Jack and the Beanstalk” on the fourth floor. Jack is shown falling as the user scrolls down. The fifth floor is space with abstract looking solar bodies are shown (use of jQuery).
The user navigates through this website by first clicking on the blinking button to call for the elevator and then once the user is inside, they can access each floor by clicking on the level number. There is a navigation bar present on top of each composition that allows the user to go to another floor without starting the process all over.
