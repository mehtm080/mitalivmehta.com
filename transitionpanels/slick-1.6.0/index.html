The space that I selected was my dormitory lobby which is on 65 5th Avenue, New York. I thoroughly documented this site by photographing the different features present here before building it using HTML, CSS and jQuery. This is an abstract way of showing the interior layout and its functions. The cursor is The New School student ID card. The darker boxes are the classrooms on top and the street to the left. Hovering over changes the colors of the shapes and this provides an interactive element to the piece. The way the user navigates my website relates to the way one moves through this space.
